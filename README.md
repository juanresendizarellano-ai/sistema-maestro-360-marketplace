# Sistema Maestro 360 - Marketplace
Professional AI-powered business skills for SMBs. Install via Claude Code:
```
/plugin marketplace add juanresendizarellano-ai/sistema-maestro-360-marketplace
/plugin install meta-ads-reporter@sistema-maestro-360
```
## Plugins
### Meta Ads Reporter v1.0.0
Transform Meta Ads CSV exports into professional KPI dashboards (Excel) and written reports (Word) with efficiency scores, campaign rankings, automated insights, and period comparisons.
**Features:** KPI calculation, efficiency scoring, Top 3 best/worst campaigns, period comparison, automated insights, benchmarking.
